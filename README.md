## Features

- Vue 3
- TypeScript
- Tailwind CSS
- Simple store with global state
- Vue Router
- ESLint

## Project setup and usage

Install dependencies:

```
npm install
```

Run development server:

```
npm run dev
```

Build for produciton:

```
npm run build
```

Run ESLint:

```
npm run lint
```

Lint and fix:

```
npm run lint:fix
```
